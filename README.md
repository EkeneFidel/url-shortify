## Getting started

1. Clone the repository

```console
git clone https://github.com/EkeneFidel/url-shortify
```

2. Install the dependencies

```console
npm install
```

3. Run the application

```console
npm start:app
```

4. Access application using this link

```console
http://localhost:3000/
```
